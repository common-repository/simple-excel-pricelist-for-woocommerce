=== Simple Excel Pricelist for WooCommerce ===
Tags: woocommerce, pricelist, excel
Requires at least: 4.0
Tested up to: 5.4
License: GPLv2 or later

This plugin helps to create a price list of all products available in stock in excel format and allows users
to download the file.

== Installation ==

Install and activate the plugin via your WordPress Administration panel.
